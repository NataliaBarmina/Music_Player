# Music Player

Музыкальный плеер на React, TypeScript и Vite.

## Описание

Music Player — это веб-приложение для прослушивания музыки и работы с плейлистом.

В проекте реализуется интерфейс музыкального плеера: отображение треков, управление воспроизведением и работа со списком композиций.

## Основная функциональность

- отображение списка треков;
- воспроизведение музыки;
- пауза и продолжение воспроизведения;
- переключение треков;
- работа с текущим треком;
- изменение порядка треков в плейлисте.
